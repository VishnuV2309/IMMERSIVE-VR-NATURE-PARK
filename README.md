Immersive VR Nature Park

A Web-Based VR Experience Using A-Frame
Developed by VISHNU V (R23EI051)

Overview

Immersive VR Nature Park is a browser-based VR experience that simulates a calm, natural environment with trees, animals, water, fire, and ambient spatial audio.
Users can explore the 3D world using teleportation, interact with objects, and experience animations and soundsâ€”all built using A-Frame, HTML, and JavaScript.

Features

Fully interactive environment (trees, ducks, rabbits, deer, water, fire).

Teleportation-based movement.

Procedurally generated trees with sway animation.

Ambient background audio + object sound effects.

Browser-based VR with WebXR support.

Lightweight and works on desktop, laptop, and some mobile devices.

How to Run

Keep all files (HTML + MP3s) in the same folder.

Open index.html in Chrome or Edge.

Click once to enable audio.

Look around with mouse; click ground to teleport; click objects to interact.

No installation or internet required after downloading the files.
